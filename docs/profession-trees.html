Stat Bars: Each weapon shows 6 normalized bars (0–100) for Damage, Speed, Combo, Crit, Block, Defense. Abilities: Basic Attack + Signature + 3 Attacks + 3 Spells. Passives: 3 per weapon, unlocking at T5, T6, and T8. Example — Bloodfeud Blade: DMG 75 · SPD 60 · COMBO 85 · CRIT 50 · BLK 40 · DEF 30. Passives: Bloodlust (T5, +5% atk speed on kill), Swift Vengeance (T6, crits reduce CD), Deep Cuts (T8, +25% bleed dmg).
