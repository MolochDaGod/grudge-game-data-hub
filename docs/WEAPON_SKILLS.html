    VFX Asset Pipeline

    Each skill references VFX assets via path keys that resolve against the CDN:

    • attackAnimation — Mixamo/FBX animation clip name (e.g. 1h_sword_slash_01)
    • slashVfx — Slash trail particle (vfx/slash/sword_arc.png)
    • projectileVfx — Projectile particle system (vfx/projectile/fire_bolt.glb)
    • projectileAsset — 3D model for projectile (models/projectiles/arrow.glb)
    • impactVfx — Hit impact particle (vfx/impact/blood_splash.png)
    • aoeVfx — Area-of-effect ground decal/particle (vfx/aoe/fire_circle.glb)
    • teleportVfx — Teleport/blink particle (vfx/teleport/shadow_phase.png)
    • comboOrder — Array of animation clip names for combo chain sequence

    Best Practices

    • Canvas sizing: ResizeObserver on container. renderer.setPixelRatio(Math.min(devicePixelRatio, 2)).
    • VFX sprites: SpriteMaterial with additive blending for fire/magic. PointsMaterial for particles.
    • Animation blending: clipAction(clip).crossFadeTo(next, 0.15) for smooth combo transitions.
    • Combo chains: mixer.addEventListener('finished') to advance to next step in comboOrder.
    • Projectile pooling: Pre-instantiate meshes. Object pool — acquire() / release().
    • AoE decals: DecalGeometry or ground-projected PlaneGeometry with alpha + bloom.
    • R2 loading: Always from https://assets.grudge-studio.com/. Use LoadingManager for progress.
    • Dispose: geometry.dispose(), material.dispose(), texture.dispose() when removing VFX.
