Block & Critical

Block: IF Random < Block Chance → Damage = Damage × (1 - Block Factor)
Critical: IF Random < Crit Chance → Damage = Damage × Crit Factor

Note: Block check happens BEFORE crit check. Crits cannot occur on blocked attacks.

Drain / Reflect / Absorb

Health Restored = Damage Dealt × Drain Health Factor (cap 50%)
Damage Reflected = Damage Received × Reflect Factor (cap 50%)
Absorb = Damage Received × Absorb Factor (cap 50%)

Reflect does NOT proc on blocked hits.

Full Combat Pipeline (8 Steps)

  1. Calculate Base Damage — Level + Attributes + Equipment + Traits
  2. Apply Defense Break — Reduce target defense by attacker's Defense Break Factor
  3. Calculate Mitigation — Apply defense formula (√defense reduction)
  4. Apply Random Variance — ±25% deviation (if enabled)
  5. Check Block — Roll vs effective block chance (modified by Block Break)
  6. Check Critical — Roll vs effective crit chance (modified by Crit Evasion) if not blocked
  7. Apply Damage — Subtract from target HP
  8. Trigger Effects — Drain, Reflect, Absorb factors activate
