Block & Critical

Block: IF Random < Block Chance → Damage = Damage × (1 - Block Factor)
Critical: IF Random < Crit Chance → Damage = Damage × Crit Factor

Note: Block check happens BEFORE crit check. Crits cannot occur on blocked attacks.

Drain / Reflect / Absorb

Health Restored = Damage Dealt × Drain Health Factor (cap 50%)
Damage Reflected = Damage Received × Reflect Factor (cap 50%)
Absorb = Damage Received × Absorb Factor (cap 50%)

Reflect does NOT proc on blocked hits.

Full Combat Pipeline (8 Steps)

  1. Calculate Base Damage — Level + Attributes + Equipment + Traits
  2. Apply Defense Break — Reduce target defense by attacker's Defense Break Factor
  3. Calculate Mitigation — Apply defense formula (√defense reduction)
  4. Apply Random Variance — ±25% deviation (if enabled)
  5. Check Block — Roll vs effective block chance (modified by Block Break)
  6. Check Critical — Roll vs effective crit chance (modified by Crit Evasion) if not blocked
  7. Apply Damage — Subtract from target HP
  8. Trigger Effects — Drain, Reflect, Absorb factors activate

Tier System (T1-T8)

T5 renamed from "Legendary" to Heroic; T8 renamed from "Legendary Artifact" to Legendary. The word Legendary now refers to the T8 tier label only. The Artifact weapon category (end-game, world-found) is separate - see the Items & System Guide for details.

Elemental System

7 element types. Each weapon ability carries an element; magical damage is mitigated by the target's resistance to that element. Resistance caps at 75%, but can go negative (-50%) for vulnerability. Source: combatCalculations.ts.

🔥 Fire

Burn stacks, AoE damage. Fire Staves primary element.

❄️ Ice

Chill, slow, deep freeze. Frost Staves primary element.

⚡ Lightning

Chain damage, stun, burst. Lightning Staves primary element.

🌀 Arcane

Raw magic damage. Artifact-arcane sub-type.

✨ Holy

Heals, cleanses, divine shields. Holy Staves primary element.

🌿 Nature

HoT, reflect, entangle. Nature Staves primary element.

⚔ Physical

Default for melee weapons. Mitigated by physical defense.

Resistance Formula:
Magical Damage = Base Magic × (1 - ElementResist / 100)
Physical Damage = Base Phys × (1 - PhysResist / 100)
Final = (PhysDmg + MagDmg) / 2 after individual resist application

Cap: 75% · Floor: -50% (vulnerability)

Weapon Abilities & Procs

Each weapon grants abilities: Basic Attack, Signature, 3 Attacks, 3 Spells, and Passives. Abilities carry element affinity, damage multipliers, mana costs, cooldowns, and triggered effects. Source: weaponArsenal.ts, combatCalculations.ts.

Buffs

Temporary stat bonuses with duration. Stacks from abilities, potions, or passives. Each buff has statBonus[] with stat key + value.

Debuffs

Negative stat effects on targets. Include bleeds, slows, armor breaks. Each debuff has statPenalty[] with stat key + value.

Combo State

Tracks comboCounter, lastAbilityUsed, canCombo. Combos trigger when matching the weapon's combo chain order. Worge has a unique counterTarget mechanic.

Passives

Unlock at T5+. Up to 3 per weapon. Provide persistent stat bonuses, on-hit effects, or conditional triggers (on block, on crit, on kill).
